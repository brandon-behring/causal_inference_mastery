## Technical Decisions

### Estimators to Implement

**Core Estimators**:
1. **2SLS** - Workhorse, easy to understand, fast
2. **LIML** - Less biased under weak IV
3. **GMM** - Overidentification tests, HAC robustness

**Weak IV Robust**:
4. **Anderson-Rubin** - Always valid, conservative
5. **Conditional LR** - More powerful than AR

**Not implementing** (future work):
- Fuller k-class estimators (marginal improvement over LIML)
- JIVE (jackknife IV) - niche use case
- Split-sample IV - rarely used

### Diagnostic Priorities

**Critical Diagnostics** (must implement):
- F-statistic from first stage
- Cragg-Donald statistic
- Stock-Yogo critical values
- Sargan/Hansen J overidentification test

**Nice-to-Have** (defer):
- Kleibergen-Paap rk statistic (clustered standard errors)
- Montiel Olea-Pflueger effective F (many instruments)

## Performance Targets

**2SLS**: O(n * K²) where K = #instruments
- n=1,000, K=10: < 50 ms
- n=10,000, K=10: < 500 ms

**LIML**: O(n * K² + K³) (eigenvalue computation)
- n=1,000, K=10: < 100 ms
- n=10,000, K=10: < 1 second

**GMM**: O(iterations * n * K²)
- n=1,000, K=10: < 200 ms
- n=10,000, K=10: < 2 seconds
