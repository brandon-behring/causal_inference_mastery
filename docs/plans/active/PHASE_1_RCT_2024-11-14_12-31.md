## Decisions Made

### 2024-11-14: Test-First from Day 1
**Decision**: Write all tests before implementation
**Reason**: Known-answer tests catch subtle bugs, force clear thinking about edge cases
**Alternative**: Implement then test
**Impact**: Slower initial progress but higher confidence in correctness

### 2024-11-14: Monte Carlo with 1000 Simulations
**Decision**: Use 1000 simulations for validation (not 100 or 10000)
**Reason**: 1000 gives stable estimates without excessive runtime
**Impact**: Each validation run takes ~30 seconds
